电动力学

课程简介

电动力学是研究电磁场的基本属性、运动规律以及电磁场与带电物质之间相互作用的学科。本课程主要学习麦克斯韦方程组、电磁波的传播、辐射以及电磁场与物质的相互作用等内容。

注：所有试题答案一律写在答题纸上，答案写在试卷、草稿纸上一律无效。

一、填空题（共26分，共13个空，每空2分）

  1. 麦克斯韦总结了实验定律：____、____、____和____，引入了____假说，得出了电磁场满足的方程。

  2. 真空中静止的点电荷Q对另一个静止点电荷Q'的作用力F̄为____。

  3. 导体内的自由电荷分布为____，只要电磁波满足____就可认为____，此时导体可视为良导体。

  4. 电磁波在真空中的波动方程为____和____。

  5. 真空中有一半径为R₀的接地导体球，距球心为a(a > R₀)处有一点电荷Q，则像电荷的位置在距球心____处，像电荷大小为____。

二、简答题（共20分，共4小题，每小题5分）

  1. Ē、D̄、B̄和H̄这几个物理量中哪两个量是基本量？为什么？写出辅助量和基本量的关系。

  2. 为什么电磁场的Ē和B̄并不对应唯一的矢势和标势？对应同一电磁场的势满足什么条件？

  3. 用电磁辐射理论解释旭日东升为红色，晴空为蓝色。

  4. 用标势求静电场和静磁场过程有什么区别？在电磁场中的矢势和标势与静电场静磁场中的势有什么区别？

三、证明题（共14分，共2小题，每小题7分）

  1. 证明均匀介质内部，极化电荷体密度ρp总等于自由电流ρf的-(1 - ε₀/ε)倍。

  2. 证明：当两绝缘介质的分界面不带自由电荷时，电场线的曲折线满足

    tan θ₂ / tan θ₁ = ε₂ / ε₁

    其中ε₁和ε₂分别为两种介质的介电常数，θ₁和θ₂分别为界面两侧电场线与法线的夹角。

四、计算题（共40分，共3小题）

  1. （10分）设真空中的电势为

    φ(r) = Q/(4πε₀r)     (r > a) Q/(4πε₀a)     (r ≤ a)

    求自由电荷分布。

  2. （15分）半径为R₀的带电导体球面，面电荷密度为σ = σ₀cosθ，球外充满介电常数为ε的均匀介质，求球内外的电势和电场强度。

  3. （15分）由麦克斯韦方程组推导标势φ和矢势Ā所满足的基本方程组；并在洛伦兹规范下，得出用势表示的电动力学基本方程组。

电动力学知识点总结

一、麦克斯韦方程组

麦克斯韦总结了四个实验定律，并引入了位移电流假说，得出了完整的电磁场方程组：

∇ × E = -∂B/∂t    (法拉第定律)
∇ × H = J + ∂D/∂t    (安培定律，含位移电流)
∇ · D = ρ    (高斯定律)
∇ · B = 0    (磁高斯定律)

二、基本物理量

1. 基本量与辅助量

  • 基本量：E（电场强度）和B（磁感应强度）
  • 辅助量：D（电位移矢量）和H（磁场强度）

关系式：

D = ε₀E + P = εE
H = B/μ₀ - M = B/μ

三、电磁势

1. 矢势和标势

电磁场的E和B并不对应唯一的矢势A和标势φ，对应同一电磁场的势满足规范变换：

A' = A + ∇ψ
φ' = φ - ∂ψ/∂t

2. 洛伦兹规范

在洛伦兹规范下（∇·A + (1/c²)∂φ/∂t = 0），势满足的方程为：

∇²φ - (1/c²)∂²φ/∂t² = -ρ/ε₀
∇²A - (1/c²)∂²A/∂t² = -μ₀J

四、静电场与静磁场

1. 静电场

用标势φ描述：E = -∇φ，满足泊松方程：

∇²φ = -ρ/ε₀

2. 静磁场

用矢势A描述：B = ∇×A，满足：

∇²A = -μ₀J

五、介质中的电磁场

1. 极化电荷

在均匀介质内部，极化电荷体密度ρp与自由电荷体密度ρf的关系：

ρp = -(1 - ε₀/ε)ρf

2. 边界条件

当两绝缘介质分界面不带自由电荷时，电场线的折射满足：

tan θ₂ / tan θ₁ = ε₂ / ε₁

其中θ₁和θ₂分别为界面两侧电场线与法线的夹角。

六、电磁波

1. 波动方程

在真空中，电磁波满足波动方程：

∇²E - (1/c²)∂²E/∂t² = 0
∇²B - (1/c²)∂²B/∂t² = 0

2. 导体中的电磁波

在良导体中，自由电荷分布在外表面，只要电磁波频率满足一定条件，导体可视为良导体。

七、像电荷法

对于接地导体球外一点电荷Q的问题，可以用像电荷法求解：

  • 像电荷位置：在球心与点电荷连线上，距球心R₀²/a处
  • 像电荷大小：-QR₀/a

八、电磁辐射

1. 瑞利散射

用电磁辐射理论可以解释：

  • 旭日东升为红色：短波长（蓝光）被大气散射，长波长（红光）直射
  • 晴空为蓝色：短波长（蓝光）被大气散射到各个方向

九、重要公式总结

库仑定律：F = (1/4πε₀)(QQ'/r²)
电势能：U = qφ
电场能量密度：u = ½ε₀E²
磁场能量密度：u = B²/(2μ₀)
