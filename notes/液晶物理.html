液晶物理

课程简介

液晶物理是研究液晶材料的结构、性质和应用的基础课程。液晶是介于各向异性晶体和各向同性液体之间的中间相，具有液体的流动性和晶体的各向异性。本课程主要学习液晶的分类、序参数、Freedericks转变、Frank弹性理论等内容。

一、液晶物理基础

1. 液晶的定义

液晶（Liquid Crystal）：介于各向异性晶体和各向同性液体之间的中间相。

特点：

  • 具有液体的流动性
  • 具有晶体的各向异性（anisotropy）

2. 液晶的分类

① 根据形成条件分类

  • 热致液晶（Thermotropic）：由温度变化引起的液晶相
  • 溶致液晶（Lyotropic）：由浓度变化引起的液晶相

② 根据结构分类

  • 向列相（Nematic）：分子长轴方向有序，位置无序
  • 胆甾相（Cholesteric）：螺旋结构，具有手性
  • 近晶相（Smectic）：层状结构，分子排列更有序

3. 序参数（Orientational Order Parameter）

序参数S用于描述液晶分子的取向有序程度：

S = <P₂(cos θ)> = <(3cos²θ - 1)/2>

其中：

  • θ：分子长轴与指向矢n的夹角
  • S = 1：完全有序（晶体）
  • S = 0：完全无序（各向同性液体）
  • 0 < S < 1：液晶相

4. 序参数张量

在空间固定坐标系中，序参数张量Q定义为：

Qαβ = S(nαnβ - δαβ/3)

其中n是指向矢（director），S是标量序参数。

二、Frank弹性理论

1. Frank弹性自由能密度

Frank弹性自由能密度F包含三种基本形变：

F = ½k₁₁(∇·n̂)² + ½k₂₂(n̂·∇×n̂)² + ½k₃₃(n̂×∇×n̂)²

其中：

  • k₁₁：展曲弹性常数（splay）
  • k₂₂：扭曲弹性常数（twist）
  • k₃₃：弯曲弹性常数（bend）

2. 三种基本形变

  • 展曲（Splay）：∇·n̂ ≠ 0，分子呈扇形排列
  • 扭曲（Twist）：n̂·∇×n̂ ≠ 0，分子螺旋排列
  • 弯曲（Bend）：n̂×∇×n̂ ≠ 0，分子呈弧形排列

三、Freedericks转变

1. 基本概念

Freedericks转变是液晶在外场（电场或磁场）作用下发生的指向矢重新取向的临界现象。

2. 磁场诱导的Freedericks转变

① 边界条件

假设液晶盒厚度为d，边界条件为：

θ|z=0 = 0,    θ|z=d = 0

初始状态：n̂ = (1, 0, 0)，磁场H = (0, H, 0)

② 形变状态

在磁场作用下，指向矢发生倾斜：

n̂ = (cos θ, sin θ, 0),    θ = θ(z)

③ 自由能

单位面积自由能：

A = ∫₀ᵈ [½k₂₂(dθ/dz)² - ½ΔχH²sin²θ] dz

其中Δχ是磁化率各向异性。

④ 临界磁场

通过变分原理求解，得到临界磁场：

Hc = (π/d)√(k₂₂/Δχ)

当H < Hc时，液晶保持初始排列；当H > Hc时，发生Freedericks转变。

⑤ 角度分布

在转变后，角度分布满足：

dθ/dz = √[(ΔχH²/k₂₂)(sin²θmax - sin²θ)]

四、超扭曲向列相（STN）液晶显示器

1. STN的基本原理

STN（Super Twisted Nematic）是一种扭曲角大于90°的液晶显示模式。

① 结构特点

  • 上下基板表面处理：摩擦方向相反（±45°）
  • 上下基板摩擦方向与液晶分子长轴方向一致
  • 扭曲角通常为180°-270°

② 工作原理

  • 无电场时：液晶分子呈螺旋排列，产生光学双折射效应
  • 有电场时：液晶分子沿电场方向排列，螺旋结构被破坏
  • 通过控制电场，可以实现明暗显示

2. STN的响应特性

STN的响应时间与扭曲角、液晶材料参数、驱动电压等因素有关。通常响应时间较长，但可以通过优化材料参数来改善。
