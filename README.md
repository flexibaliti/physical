# 应用物理学习笔记网站

河北工业大学应用物理系学习笔记平台

## 项目简介

这是一个静态网站项目，用于展示和分享应用物理学的学习笔记。网站包含多个科目的笔记内容、试卷和概念题集。

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- 静态网站部署

## 项目结构

```
应用物理笔记整理/
├── index.html              # 首页
├── styles/                 # 样式文件
│   ├── main.css           # 主样式
│   └── note.css           # 笔记页面样式
├── scripts/                # JavaScript文件
│   ├── main.js            # 主页脚本
│   └── note.js            # 笔记页面脚本
├── notes/                  # 笔记内容
│   ├── *.html             # 各科目笔记页面
│   └── 笔记/              # PDF和图片资源
│       ├── *.pdf          # PDF笔记文件
│       └── *.jpg          # 图片资源
└── README.md              # 项目说明
```

## 本地运行

直接在浏览器中打开 `index.html` 即可查看网站。

## 部署说明

本项目可以部署到 Netlify、Vercel、GitHub Pages 等静态网站托管平台。
